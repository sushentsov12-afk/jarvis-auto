# 🚀 Deployment Guide - Jarvis Auto\n\n## Vercel Deployment\n\n### Step 1: Prepare Firebase\n\n1. Go to [Firebase Console](https://console.firebase.google.com)\n2. Create/Select your project\n3. Get credentials:\n   - API Key\n   - Auth Domain\n   - Project ID\n   - Storage Bucket\n   - Messaging Sender ID\n   - App ID\n\n### Step 2: Google OAuth Setup\n\n1. Go to [Google Cloud Console](https://console.cloud.google.com)\n2. Create OAuth 2.0 Client ID (Web Application)\n3. Add redirect URIs:\n   - `http://localhost:5173` (dev)\n   - `https://jarvis-auto-psi.vercel.app` (production)\n4. Copy Client ID\n\n### Step 3: Deploy to Vercel\n\n```bash\n# 1. Login to Vercel\nnpm i -g vercel\nvercel login\n\n# 2. Deploy\nvercel --prod\n```\n\n### Step 4: Add Environment Variables\n\n1. Go to Vercel Dashboard → Project Settings → Environment Variables\n2. Add all variables from `.env.local`:\n   ```\n   VITE_FIREBASE_API_KEY\n   VITE_FIREBASE_AUTH_DOMAIN\n   VITE_FIREBASE_PROJECT_ID\n   VITE_FIREBASE_STORAGE_BUCKET\n   VITE_FIREBASE_MESSAGING_SENDER_ID\n   VITE_FIREBASE_APP_ID\n   VITE_GOOGLE_CLIENT_ID\n   ```\n\n### Step 5: Firebase Security Rules\n\nUpdate `firestore.rules` in Firebase Console:\n\n```\nrules_version = '2';\nservice cloud.firestore {\n  match /databases/{database}/documents {\n    match /users/{uid} {\n      allow read, write: if request.auth.uid == uid;\n    }\n    match /diagnoses/{uid}/history/{document=**} {\n      allow read, write: if request.auth.uid == uid;\n    }\n    match /cars/{uid}/list/{document=**} {\n      allow read, write: if request.auth.uid == uid;\n    }\n  }\n}\n```\n\n## Local Development\n\n```bash\n# Install dependencies\nnpm install\n\n# Create .env.local with your credentials\ncp .env.example .env.local\n\n# Start dev server\nnpm run dev\n\n# Build for production\nnpm run build\n\n# Preview production build\nnpm run preview\n```\n\n## Troubleshooting\n\n### \"Firebase API Key is invalid\"\n- Check your Firebase credentials in `.env.local`\n- Ensure API Key has proper permissions\n\n### \"Google OAuth failed\"\n- Verify Client ID is correct\n- Check redirect URIs in Google Cloud Console\n- Clear browser cookies and cache\n\n### \"Data not syncing\"\n- Check Firestore Rules\n- Verify user is authenticated\n- Check browser console for errors\n\n## Monitoring\n\n- Vercel Dashboard: https://vercel.com/dashboard\n- Firebase Console: https://console.firebase.google.com\n- Google Cloud Console: https://console.cloud.google.com\n"
