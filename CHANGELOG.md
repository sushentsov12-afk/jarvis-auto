# Changelog — Jarvis Auto

Все значимые изменения по версиям.

---

## [8.0.0] — 2025-05 · ТЕКУЩАЯ

### Добавлено
- 📊 **Аналитика** — отдельный экран: стоимость/км, расход топлива, тренды по месяцам, прогноз ближайших ТО
- 🔔 **Центр уведомлений** — единое окно всех алертов: просрочка ТО, истекающие документы, давление шин, предстоящие записи
- ⚡ **Быстрый расход** — bottom sheet для добавления трат в 2 касания прямо с главного экрана
- 🔍 **Поиск по истории** — диагнозы и расходы теперь фильтруются по тексту
- 📅 **Центр записей** — счётчик предстоящих визитов в сервис на главном экране
- 🛞 **Уведомления о давлении** — низкое/критичное давление отображается в центре уведомлений
- 📄 **Счётчик ОСАГО** — виджет на главном экране с отсчётом дней
- 💡 **Авто-советы** — каждые 30 минут появляется всплывающий совет Джарвиса

### Улучшено
- Главный экран: приветствие по имени, быстрые действия, панель из 5 метрик
- История диагнозов: граф по уровням опасности в модуле аналитики
- Сервисная книжка: вкладка «История замен» с хронологией всех работ
- Все 7 вкладок теперь со счётчиками проблем на иконках

## [7.0.0] — 2025-04

### Добавлено
- 📄 Модуль документов — ОСАГО, КАСКО, техосмотр, права с датами истечения
- 📅 Записи в сервис — планирование визитов с датой, временем, телефоном
- 📍 Геолокация — определение региона через Nominatim, региональный КБМ ОСАГО
- 🔍 Поиск сервисов рядом через ИИ (demo-режим)
- 💡 ИИ-чат «Советы Джарвиса» — вопросы о конкретном авто
- 🆘 Экстренная помощь — ДТП, пробитое колесо, прикурка, перегрев

## [1.0.0] — 2024-11 · MVP

### Первый релиз
- HTML standalone — одностраничное приложение без сборщика
- Расшифровка кодов OBD-II через Claude API
- 4 быстрых примера (P0301, P0420, стук, не заводится)
- Результат: название, система, опасность, можно ехать / нет, цена ремонта, 3 действия
- Цветовая схема Deep Blue + Safety Orange
- Шрифты IBM Plex Mono + IBM Plex Sans
